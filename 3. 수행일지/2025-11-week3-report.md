## 1. 프로젝트 수행 계획 및 현황

### 금주 프로젝트 수행 내용

---

-   김지용
    -   DOCS : 핵심 기능 중 유저 등급 관리(신러되에 따른 리뷰 등급&뱃지 부여), 개인 피부 DB 기반 성분 추천/ 주의 정책 제작, 기획서 최신화, 멘토링 PT 준비
    -   DB : 비즈니스 구성에 맞춘 DB 수정 및 문서, DB 코맨트 최신화, 추후 BE 리뷰 이미지 넣어야할 OCI backet 준비
    -   BE : 리뷰 유저 등급관리 정책에 맞춰 구성
    -   FE : 마이페이지 구현
-   김지훈
    -   FE: 장바구니 페이지 구현 및 API 연동
    -   BE: 장바구니 기능(CRUD 및 성분 조언) 완료, 결제 API 진행 중
-   모란희

    -   BE : 조회 api 추가(주문 번호, 택배 목록, 택배 번호). api param재가공 클래스 추가. api 날짜 포맷 적용.
    -   FE : 페이지 추가(주문상세, 택배목록, 택배상세). 관리자 디렉터리 구조 변경. 관리자 메인 active 스타일 추가. 관리자 목록&상세 페이지 템플릿화. 로딩 컴포넌트 추가.
    -   문서 : api 문서 작성(주문번호, 택배목록, 택배번호). 추천/주의 성분정보 정책 작성.

-   문지용
    -   BE : 상품 관리 기능(CRUD) 완료, 성분 관리 진행중
    -   FE : 상품 관리 페이지 추가, 성분 관리 페이지 추가, 페이지 간 연결 진행중
-   박민우
    -   FE: 제품 상세페이지 구현 완료, db 연결 진행중
    -   BE: 상세페이지 구현에 필요한 백엔드 API 정리
-   윤예주
    -   FE: 카카오 로그인 Redirect + Callback 기본 동작 구현,
    -   BE:이메일 로그인·회원가입 API 연동 위한 기본 틀 세팅

### 차주 프로젝트 수행 계획

---

-   김지용
    -   FE : 마이페이지 추가 구현
    -   BE : 마이페이지 필요 api 구성, 리뷰 사진 넣기
-   김지훈
    -   FE : 결제 페이지 구현 및 API 연동
    -   BE : 결제 기능 완성
-   모란희

    -   BE : 제품&피부타입 정보 기반 추천/주의 성분 API 추가. 메인 상품추천 5개 API 추가. 사용자 주문 목록&상세내용 API 추가.
    -   FE : 사용자 주문 목록&상세내용 페이지 추가.

-   문지용
    -   BE : 성분 추가, 수정 기능 완성, 문의사항 작성/수정/삭제 기능 진행
    -   FE : 성분 추가, 수정 페이지 완성, 고객센터 페이지 진행
-   박민우
    -   FE: 제품 상세페이지 구현 및 연동 완료, 리뷰 작성창 구현
    -   BE: 빠진 API 구성(신고, 태그, 이미지 등)
-   윤예주
    -   FE:카카오 로그인 + 이메일 로그인 완전 연동 마무리
        휴대폰 인증 실제 서비스 선택 및 개발
