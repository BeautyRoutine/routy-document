## 1. 프로젝트 수행 계획 및 현황

### 금주 프로젝트 수행 내용

---

-   김지용
    -   FE : Mypage 추가 구현 및 api 연동 준비 완료
    -   BE : 마이페이지 및 검색창 필요 API 구현 알고리즘 제작
    -   DB : 필요한 테이블 추가 (유저 좋아요, 유저 피부타입), 성분 관련 테이블 수정 및 추가 완료
    -   문서 : API 및 추가 DB 관련 내용 수정 및 추가
-   김지훈
    -   FE: 결제 페이지 구현, tosspayments를 이용한 결제 연동
    -   BE: 결제 API 구현 중
-   모란희

    -   BE : 피부타입에 따른 추천 제품 목록 API. 당신을 위한 맞춤 추천 API. 주문 상세페이지 > 주문 제품 목록 API. 상품등록 API에 성분 자동매핑 추가.
    -   FE : 관리자 상품관리 라우트 정리. 상품등록 > 주요성분 입력란 추가. 관리자 주문 상세 > 주문상품 목록 추가. 관리자 주문 상세 > 택배배송 목록 추가.
    -   문서 : 상품등록 API 수정(성분 자동매핑).

-   문지용
    -   BE : 최근 본 상품에 따른 제품 추천 진행중
    -   FE : 피부타입에 따른 추천 제품 목록, 당신을 위한 맞춤 추천 목록 진행중
-   박민우

    -   FE: 리뷰 예외처리/베스트 리뷰 전략 구현, 하단 탭 개편, 제품 상세조회 백엔드 연결
    -   BE: 리뷰 이미지 api 구성, 상세조회 api 수정, 연결에 필요한 백엔드 보안 설정 수정
    -   DB: 리뷰 이미지 테이블 구성
    -   문서 : 상세조회 API 수정

-   윤예주
    로그인, 회원가입 sms, 카톡 연결 지연 해결 중
