# Routy — 종합 문서 관리 레포지토리

Routy는 조직의 문서(설계서, WBS, 수행일지, 포트폴리오 등)를 중앙에서 관리하고 버전화하며, 검색과 재사용을 쉽게 하기 위한 종합 문서 관리 레포입니다. 개발자·관리자·기획자 등 다양한 역할이 문서를 일관성 있게 작성하고 유지할 수 있도록 구조와 규칙을 제공합니다.
